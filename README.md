<h1 align="center">chat 💬 PHP </h1>

## Description:
Самый обычный и простой приватный чат построенный на PHP - long-polling
Минималистичный дизайн для работы в браузере и мобильном браузере
Удобство и безопасность. Очистка - это удаление сообщений и переданных файлов на самом сервере (никаких теневых копий)
Вход в чат по паролю, пароль раздает владелец чата
Минимальное количество сообщений от 50 (можно выставить любое)
Но все сообщения перезатираюся, нигде не хранятся

## Installation:
Все файлы копируются в папку на сервер и при заходе по URL будет стартовая странциа чата
для функции sendFile() файла chat.php внтури нужно поменять путь к вашей папке на сервере (для возможности загрузки файлов)

Приложение находится в тестовой работе и разработке
